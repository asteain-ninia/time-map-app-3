# 現状.md — 実装ステータス（2026-03-22 時点）

## 概要

本アプリケーション「地物時空変遷編纂機」は初期実装フェーズにある。
プロジェクト基盤（ビルド環境・ドメインモデル・UIスケルトン）は構築済みだが、アプリケーション層以降の機能実装はまだ行われていない。

- **Svelte 5 + XState v5**: `@xstate/svelte` は Svelte 3/4 向けのため不使用。XState v5 を Svelte 5 の Runes API で手動統合する必要がある。
- **electron-vite 5**: vite 6 + @sveltejs/vite-plugin-svelte 5 との互換性のために v5 を使用。
- **ベースマップのスケーリング**: base-map.svg の viewBox (42434×21217) を SVG 上で (360×180) にスケーリングして表示。`transform="scale()"` を使用。
